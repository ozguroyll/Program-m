# 🎯 STOK YÖNETİM SİSTEMİ - EXCEL TABANLI

## 📋 PROJE AÇIKLAMASI

Bu proje, Özgür Yılmaz için özel olarak geliştirilmiş kapsamlı bir Excel tabanlı stok yönetim sistemidir. 4 şirket ve 2 Iraklı ortak için çok lokasyonlu depo yönetimi, araç bazlı sevkiyat takibi ve detaylı finansal raporlama özelliklerini içerir.

## 🚀 HIZLI BAŞLANGIÇ

### 1. Dosyaları İndirin
```bash
git clone https://github.com/[username]/Stok_Yonetim_Sistemi.git
```

### 2. Excel Dosyasını Açın
- `Stok_Yonetim_Sistemi_TAMAMLANDI.xlsx` dosyasını açın
- Makroları etkinleştirin

### 3. VBA Kodlarını Ekleyin
- Alt+F11 → VBA editörünü açın
- `VBA_Kodlari_Tam.txt` dosyasındaki kodları kopyalayın
- Yeni modül oluşturup kodları yapıştırın
- `Form_Kodlari_Tam.txt` dosyasındaki form kodlarını da ekleyin

### 4. Sistemi Başlatın
- F5 → "SistemBaslat" makrosunu çalıştırın
- "Sistem Hazır" mesajını bekleyin

1. Dashboard sekmesine gidin
2. VBA editöründe "StokGirisFormuAc" makrosunu çalıştırın
3. Form açılır → Bilgileri doldurun:
   - Ürün: Mısır
   - Miktar: 100 ton
   - Fiyat: 250 USD
   - Depo: Mersin Antrepo

4. Kaydet → Otomatik dashboard güncellenir!
