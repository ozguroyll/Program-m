# 🪟 Windows için Hububat Stok Yönetim Sistemi

## 🚀 Hızlı Başlangıç

### 1. Otomatik Kurulum
1. **setup_windows.bat** dosyasına çift tıklayın
2. Kurulum tamamlanana kadar bekleyin

### 2. Uygulamayı Başlatın
1. **start_application.bat** dosyasına çift tıklayın
2. Tarayıcınızda http://localhost:5173 adresine gidin

### 3. Uygulamayı Durdurun
1. **stop_application.bat** dosyasına çift tıklayın

## 📁 Windows Dosyaları

- **setup_windows.bat** - Otomatik kurulum scripti
- **start_application.bat** - Uygulamayı başlatma scripti
- **stop_application.bat** - Uygulamayı durdurma scripti
- **check_system.bat** - Sistem gereksinimlerini kontrol etme
- **WINDOWS_KURULUM.md** - Detaylı Windows kurulum rehberi

## 🔧 Gereksinimler

- Windows 10/11 (64-bit)
- Python 3.8+ (python.org'dan)
- Node.js 16+ (nodejs.org'dan)
- Poetry (otomatik kurulacak)

## 📊 Sistem Özellikleri

- ✅ **6 Ana Modül**: Dashboard, Stok, Cari, Gelir/Gider, Kar/Zarar, Tanımlamalar
- ✅ **Hububat Ticareti**: Kızıltepe-Irak özel iş akışı
- ✅ **Windows Uyumlu**: Tam Windows desteği
- ✅ **Otomatik Kurulum**: Tek tıkla kurulum
- ✅ **Türkçe Arayüz**: Tam Türkçe lokalizasyon

## 🚨 Sorun Giderme

1. **check_system.bat** dosyasını çalıştırarak sistem gereksinimlerini kontrol edin
2. **logs** klasöründeki hata dosyalarını inceleyin
3. Gerekli yazılımları yeniden kurun

**Windows'ta hububat ticareti için hazır! 🌾**
